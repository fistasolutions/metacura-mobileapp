# This is a recent React Native + React 19 stack

This app runs React Native 0.86 and React 19, with Reanimated 4 (worklets) and React Navigation 7. APIs, the New Architecture defaults, and worklet conventions may differ from older training data. Before writing native-bridging, animation, or navigation code, verify against the versions in `package.json` and the installed type definitions in `node_modules/`. Heed deprecation notices.

For project rules, conventions, and the Spec Sync protocol, see `CLAUDE.md`.
